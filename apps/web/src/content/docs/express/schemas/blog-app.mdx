# Blog App Schema

The Blog App Schema is a comprehensive schema for a blogging platform. It includes support for MongoDB, PostgreSQL, and MySQL databases with features like posts, categories, comments, and likes.

## Database Support

- [MongoDB(Mongoose)](/docs/express/schemas/blog-app-mongodb)
- [PostgreSQL(Drizzle)](/docs/express/schemas/blog-app-postgresql)
- [MySQL(Drizzle)](/docs/express/schemas/blog-app-mysql)

## Installation all schemas

<PackageManagerTabs command="npx servercn-cli add schema blog-app" />

## Installation individual schema

<Note text="If you install schema individually, the relationships between them won't be automatic—you'll need to implement them manually." />
