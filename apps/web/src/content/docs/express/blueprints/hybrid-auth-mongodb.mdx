# Hybrid Authentication with MongoDB (Mongoose)

MongoDB implementation of hybrid authentication using Mongoose for users and Redis for session tracking and refresh token rotation.

## Installation Guide

Add the hybrid authentication blueprint using the Servercn CLI:

<PackageManagerTabs command="npx servercn-cli add blueprint hybrid-auth" />

## API Endpoints

All endpoints are the same for both MVC and Feature architectures.
