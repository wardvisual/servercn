# Basic Bank App (Node.js)

A simple banking application built with **Node.js** that demonstrates core backend concepts such as user accounts, transactions, and balance management. This project is ideal for learning REST APIs, backend architecture, and basic financial workflows.

## Features

- Create and manage bank accounts
- Deposit and withdraw money
- Transfer funds between accounts
- View account balance and transaction history
- RESTful API design
- Input validation and error handling

## Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**
- **TypeScript**
